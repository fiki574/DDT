# Description

Classic _docker-compose.yml_ file collection for various DevOps tools.

# Recommended prerequisites

1. VM/VPS/Dedicated server
2. Ubuntu 16.04 
3. MySQL Server
4. Docker (required)
5. Docker Compose (required)
